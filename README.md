# NorthStar Urban Mobility & Logistics

Databases and Analytics Module - Coursework

## Notebooks

- `01_SQL_in_R.ipynb` - SQL queries within R (sqldf, RSQLite)
- `02_R_Analytics.ipynb` - Statistical tests and ggplot2 visualisations
- `03_Python_Data_Processing.ipynb` - Pandas/NumPy data cleaning and analysis
- `04_MongoDB_Development.ipynb` - PyMongo CRUD, aggregation, indexing

## How to Run

1. Open any notebook in Google Colab
2. Upload the CSV files from `northstar_dataset/` to the Colab session
3. Notebooks 01 and 02 need R runtime in Colab
4. Notebook 03 runs on default Python runtime
5. Notebook 04 needs a MongoDB Atlas connection string

## Tools

- R: sqldf, RSQLite, DBI, tidyverse, ggplot2, dplyr, corrplot
- Python: pandas, numpy, matplotlib, seaborn, pymongo
- Database: MongoDB Atlas, SQLite
